# 10-React-Westagram-4

대표적 소셜 네트워킹 서비스, 인스타그램을 리액트 라이브러리를 활용해 클론

 - SPA(Single Page Application)개념 이해
 - 웹 페이지의 기본적인 레이아웃 구현
 - JSX 문법을 활용한 이벤트 구현
 - git을 이용한 버전관리 및 협업
 - fetch를 사용하여 백앤드에 api를 호출하고, 응답받은 데이터를 활용(회원가입, 로그인, storage에 token 저장)
